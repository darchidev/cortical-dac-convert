Cortical-DAC Convert API

Convert CL1 H5 recording files to JSON

POST /api/convert Convert H5 to JSON

Upload a .h5 recording file from the CL1 device and receive a JSON response with decoded samples, channel count, and metadata attributes.

ParameterTypeDescription
file*File (.h5)H5 recording file from CL1 device

Example request

curl -X POST https://cortical-dac-convert.vercel.app/api/convert \
  -F "file=@recording.h5"

Response

200 OK — JSON with samples, channels, frames, and H5 attributes 

{
  "id": "conv_1749123456_abc123",
  "channels": 16,
  "frames": 48000,
  "attributes": {
    "sample_rate": 25000,
    "device_id": "CL1-12345"
  },
  "samples": [0, -1, 2, ...]
}
